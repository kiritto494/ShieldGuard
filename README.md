# ShieldGuard — Антивирус

Приложение для поиска троянов, криптомайнеров и подозрительного ПО.

## Возможности

* **Сканирование папки** — проверка выбранной директории (каждый файл по содержимому).
* **Полное сканирование** — сканирование всех дисков, тщательная проверка файлов.
* **Быстрое сканирование** — проверка временных папок и типичных мест заражения, меньше файлов — быстрее результат.

## Запуск

УСТАНОВИТЕ Python 3.8 +
и установите патчи через build\_standalone и чтобы запустить найдите в папке main.py всё наслаждайтесь использованием антивируса  :)

# ShieldGuard - Antivirus

Application for searching for Trojans, cryptominers and suspicious software.

## Features

* **Folder scan** - check the selected directory (each file by content).
* **Full scan** - scan all disks, thorough check of files.
* **Quick scan** - check temporary folders and typical infection places, less files - faster result.

## Launch

INSTALL Python 3.8 +
and install patches via build\_standalone and to run find in the folder main.py everything enjoy using the antivirus :)
